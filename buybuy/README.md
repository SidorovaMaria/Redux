## Features

**User authentication**

- Sign Up, Log Inm Log out using Firebase Authentication
- Admin and Regular User Roles

**Product Management:**

- Admin can add, edit, delete product
- Product details:
  - name
  - description
  - price
  - images
  - category

**Product Listings**

- View All products
- Sort and filter by category, price, etc
- Search Functionality

**Cart**

- Add items to the cart
- View the cart
- Update quantities/ remove items

**Responsive Deisgn**

##Tech Stack

- React(Vite)
- Redux
- Tailwinds CSS
- React Router
- MongoDB
- Firebase Auth
